# mélange crews

mélange crews is a Human-Computer Interaction (HCI) group project that focuses on designing and developing a service marketplace website. The platform helps users find and book workers for everyday needs, such as drivers, plumbers, AC technicians, electricians, babysitters, tailors, chefs, and gardeners.

This project emphasizes user experience, ease of navigation, clear information structure, and a simple booking flow from choosing a worker to completing payment.

In daily life, people often need help from service workers, but finding trusted workers quickly can be difficult. mélange crews was created to provide a simple and user-friendly platform where users can browse available workers, view their profiles, book services, and complete transactions in one place.

As an HCI project, this website focuses on how users interact with the interface, including navigation, visual hierarchy, accessibility, consistency, and ease of use.

## Features

- Home page with carousel and top worker recommendations
- Services page with search and category filter
- Worker profile page with rating and worker details
- Booking page for selecting date, time, location, and notes
- Payment page with payment method options
- Payment success confirmation page
- Transaction history page
- Responsive navigation bar with hamburger menu
- Footer with website links and social media icons

## Technologies Used

- HTML
- CSS
- JavaScript

## HCI Aspects

This project applies several Human-Computer Interaction principles:

- Consistency: Similar navbar, footer, colors, and layout are used across pages.
- Visibility: Important actions such as Book Now, Continue, and Payment Confirmation are clearly displayed.
- User Control: Users can go back, review booking details, and confirm payment before continuing.
- Simplicity: The interface is designed to be clean and easy to understand.
- Feedback: The success page gives clear confirmation after payment is completed.
- Navigation: The website provides clear navigation between Home, Services, Transaction, and Profile pages.
